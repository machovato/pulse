# Writing Voice Profile

## About This File

This file is your behavioral contract with the LinkedIn skill. Every writing decision defers to what's in here. Without it, the skill defaults to generic AI-sounding output — which is exactly what this file prevents.

**You must customize this file before running `/pulse-linkedin` for the first time.** Replace every section below with your own examples, principles, and anti-patterns. The more specific you are, the more the output sounds like you.

---

## Voice Principles

Write 5-7 principles that describe how you write. Not what you write *about* — how you *write*. Focus on structure, rhythm, and reasoning style.

Examples of good principles (replace these with your own):

- I lead with the problem, not the solution
- I use concrete examples instead of abstract claims
- I write short paragraphs — if it's longer than 3 sentences, it needs a reason
- I challenge conventional wisdom when I have evidence, not for shock value
- I end with a question or an implication, not a summary
- I avoid hedging language ("I think", "perhaps", "it seems like")
- I connect ideas across disciplines — I don't stay in one lane

## What I Don't Write

List the patterns you actively avoid. This is as important as what you *do* write — it prevents the skill from drifting toward generic LinkedIn content.

Defaults (keep these unless they don't apply to you):

- Listicles with generic takeaways ("5 things I learned from...")
- Inspirational quotes with no context
- Self-congratulatory milestone posts ("Excited to announce...")
- Vague thought leadership ("AI is changing everything")
- Long preambles before the actual point
- Anything that starts with "In today's rapidly evolving..."
- Bullet-point posts disguised as insights

## AI Guardrails

These patterns get flagged and rewritten during the voice pass. Do not remove this section.

| If the draft contains... | Replace with... |
|---|---|
| "In today's..." | Cut entirely |
| "It's worth noting..." | Just state the thing |
| "At the end of the day..." | Cut entirely |
| "Leverage" | A specific verb |
| "Utilize" | "use" |
| "Ensure" | "make sure" or cut |
| "Robust" | What specifically makes it strong? |
| "Comprehensive" | What specifically does it cover? |
| "Innovative" | Describe the innovation |
| "Excited to share..." | Cut. Start with the insight. |
| "I'm thrilled to announce..." | Cut. Start with the insight. |
| Any sentence that could appear in a press release | Rewrite in first person |

If the finished post could have been written by any professional in your field, the voice pass failed. Rewrite.
